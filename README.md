## bash-completion

![](https://github.com/fixploit03/bash-completion/blob/main/goto.gif)

Skrip untuk mengaktifkan bash completion di shell interaktif.

## Instalasi 

```sh
apt-get update
apt-get install git
git clone https://github.com/fixploit03/bash-completion.git
cd bash-completion
chmod +x instal.sh
./instal.sh
```
